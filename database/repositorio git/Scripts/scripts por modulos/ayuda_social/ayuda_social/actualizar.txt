Explica: actualiza un registro identificado por su clave primaria.
SQL:
UPDATE ayuda_social
SET ticket_interno = ?,
    id_beneficiario = ?,
    id_usuario = ?,
    id_tipo_ayuda_social = ?,
    id_solicitud_ayuda_social = ?,
    id_estado_solicitud = ?,
    tipo_ayuda = ?,
    solicitud_ayuda = ?,
    fecha_ayuda = ?,
    descripcion = ?,
    estado = ?
WHERE id_ayuda = ?;
