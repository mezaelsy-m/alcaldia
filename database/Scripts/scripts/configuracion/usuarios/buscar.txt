Explica: consulta un registro puntual por clave primaria y permite listar registros para gestion.
SQL (detalle por id):
SELECT id_usuario,
       id_empleado,
       id_dependencia,
       usuario,
       password,
       rol,
       estado
FROM usuarios
WHERE id_usuario = ?;

SQL (listado):
SELECT id_usuario,
       id_empleado,
       id_dependencia,
       usuario,
       password,
       rol,
       estado
FROM usuarios
WHERE estado = 1
ORDER BY id_usuario DESC;
