Explica: consulta un registro puntual por clave primaria y permite listar registros para gestion.
SQL (detalle por id):
SELECT id_comunidad,
       nombre_comunidad,
       estado,
       fecha_registro,
       hora_registro_12h
FROM comunidades
WHERE id_comunidad = ?;

SQL (listado):
SELECT id_comunidad,
       nombre_comunidad,
       estado,
       fecha_registro,
       hora_registro_12h
FROM comunidades
WHERE estado = 1
ORDER BY id_comunidad DESC;
