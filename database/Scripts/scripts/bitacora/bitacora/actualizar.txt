Explica: actualiza un registro identificado por su clave primaria.
SQL:
UPDATE bitacora
SET id_usuario = ?,
    tabla_afectada = ?,
    accion = ?,
    id_registro = ?,
    resumen = ?,
    detalle = ?,
    datos_antes = ?,
    datos_despues = ?,
    usuario_bd = ?,
    ipaddr = ?,
    moment = ?,
    fecha_evento = ?,
    estado = ?
WHERE id_bitacora = ?;
