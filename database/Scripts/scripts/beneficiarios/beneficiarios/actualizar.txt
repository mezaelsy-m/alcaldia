Explica: actualiza un registro identificado por su clave primaria.
SQL:
UPDATE beneficiarios
SET nacionalidad = ?,
    cedula = ?,
    nombre_beneficiario = ?,
    telefono = ?,
    id_comunidad = ?,
    comunidad = ?,
    fecha_registro = ?,
    hora_registro_12h = ?,
    estado = ?
WHERE id_beneficiario = ?;
